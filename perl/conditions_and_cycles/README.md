# Conditions and Cycles

## IF

```
if    ( EXPR ) { ... }
```
```
if    ( EXPR ) { ... }
else           { ... }
```
```
if    ( EXPR ) { ... }
elsif ( EXPR ) { ... }
else           { ... }
```

## UNLESS

```
unless ( EXPR ) { ... } # if ( not EXPR )
```
```
unless ( EXPR ) { ... }
else            { ... }
```
```
unles  ( EXPR ) { ... }
elsif  ( EXPR ) { ... }
else            { ... }
```
